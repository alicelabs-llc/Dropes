# Contributing to Dropes

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Commit: `feat: your description`
4. Push and open a Pull Request

## Development

```bash
git clone https://github.com/alicelabs-llc/Dropes.git
cd Dropes
npm install
npm run dev
```
